# google_keep_clone_backend
Google Keep Clone's backend, written in Typescript using Postgresql with Typeorm
You can find the frontend repository [here](https://github.com/pioyi/google_keep_clone_frontend).

## Built With

* [Typescript](https://www.typescriptlang.org/) - A strict syntactical superset of Javascript
* [Node.js](https://nodejs.org/en/) - Server
* [Json Web Tokens](https://jwt.io/) - Authentication / Authorization
* [Bcrypt](https://www.npmjs.com/package/bcryptjs) - Password Hashing
* [Postgresql + Typeorm](https://typeorm.io/) - SQL Database
* [Class Validator](https://github.com/typestack/class-validator) - User Input Validatiion
